# ZBot

## Overview
A simple automated product availability monitoring and purchasing assistant designed to track and notify users of inventory updates.

## Features
- Real-time inventory monitoring for GPUs and gaming consoles
- Automated purchase alerts when products are in stock
- Simple, intuitive setup with minimal configuration
- Free and open-source architecture

## Technology
- WebClient-based monitoring system
- Efficient polling mechanisms
- Extensible bot framework

## Purpose
This tool allows users to track inventory for high demand products.
